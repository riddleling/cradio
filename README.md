# CRadio

在終端機中收聽線上廣播電台。

![image](image.png)

## 系統需求

- Go 1.20+ (如要自行編譯原始碼)
- mpv (必要)

CRadio 使用 mpv 作為子行程播放線上廣播電台，所以需要先安裝 mpv。

### 安裝 mpv

- Windows: 在 Microsoft Store 裡安裝 [mpv (Unofficial)](https://apps.microsoft.com/detail/9P3JFR0CLLL6?hl=neutral&gl=TW&ocid=pdpshare)。
